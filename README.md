# npm-module-helper

### 功能说明
1. 查看项目依赖的安装版本
2. cnpm 安装项目依赖
3. 安装/卸载模块

### 效果预览
文件资源管理器  
![查看项目依赖的安装版本](./preview.png)  
package.json  
![安装卸载模块](./preview2.png)

### 使用说明
1. 选择文件夹或 package.json 文件
2. 鼠标右键打开右键菜单
3. 选择菜单项
